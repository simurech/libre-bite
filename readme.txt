WooCommerce-Erweiterung für Gastronomie: Standortverwaltung, Kanban-Dashboard, POS-System und mehr.

== Description ==

**Libre Bite** verwandelt Ihren WooCommerce-Shop in ein vollständiges Bestellsystem für Gastronomie und Lieferdienste.

### Funktionen (Kostenlos)

*   **Standortverwaltung**: Einfache Verwaltung von Adressen und Öffnungszeiten.
*   **Kanban-Dashboard**: Real-time Bestellübersicht für die Küche (Eingang -> Zubereitung -> Fertig).
*   **POS / Kasse**: Einfache Oberfläche für Walk-in Kunden.
*   **Bestell-Optionen**: Zeitwahl (Sofort/Später) für Kunden.
*   **Produkt-Optionen**: Einfache Verwaltung von Extras.

### Premium Features

*   **Multi-Standort**: Verwalten Sie mehrere Filialen gleichzeitig.
*   **Erweiterte Benachrichtigungen**: E-Mail-Reminder und mehr.
*   **Optimierter Checkout**: Spezieller Checkout für Food-Ordering.
*   **Trinkgeld-System**: Integrierte Trinkgeld-Funktion.

== Installation ==

1.  Laden Sie das Plugin hoch in den Ordner `/wp-content/plugins/`.
2.  Aktivieren Sie das Plugin im Menü 'Plugins' in WordPress.
3.  Stellen Sie sicher, dass WooCommerce installiert und aktiviert ist.
4.  Konfigurieren Sie Ihre Standorte unter 'Libre Bite -> Standorte'.

== Frequently Asked Questions ==

= Benötige ich WooCommerce? =
Ja, dieses Plugin ist eine Erweiterung für WooCommerce und benötigt dieses zwingend.

= Ist das Plugin kompatibel mit HPOS? =
Ja, Libre Bite unterstützt High-Performance Order Storage (HPOS).
